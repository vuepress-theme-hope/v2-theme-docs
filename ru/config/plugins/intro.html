                              • Официальный плагин

                            vuepress-theme-hope объединяет множество плагинов VuePress.

                            • Некоторые плагины включаются автоматически, вы можете отключить их в настройках темы, если они вам не нужны.
                            • Некоторые плагины включаются только тогда, когда вы указываете необходимые параметры.

                            Заметка

                            Как член VuePress Orgopen in new window, Mr. Hope разработал множество плагинов VuePress.

                            Все плагины, называемые vuepress-theme-hope, являются официальными плагинами или плагинами, разработанными г-ном Хоупом, которые размещены в репозитории vuepress-theme-hope.

                            • Документацию по официальному плагину см. на официальном сайте VuePress2open in new window
                            • Все плагины, разработанные Mr.Hope, имеют собственную документацию и могут использоваться с другими темами.

                            # Опции плагина

                            Тема предоставляет опцию plugins для передачи параметров плагинам.

                            Имя параметров плагина

                            Все имена ключей в опции plugins представляют собой версию имени плагина в верблюжьем регистре, с удаленным необязательным суффиксом цифры 2.

                            Например:

                            • vuepress-plugin-copy-code2 управляется именем ключа copyCode.
                            • vuepress-plugin-md-enhance2 управляется именем ключа mdEnhance.

                            # Список плагинов

                            # Плагины предоставлены Mr.Hope

                            • vuepress-plugin-auto-catalog: Catalog automatically generation for VuePress2
                            • vuepress-plugin-blog2: Плагин блога для VuePress2
                            • vuepress-plugin-comment2: Обеспечивает функцию комментариев и просмотра страниц
                            • vuepress-plugin-components: Предоставляет некоторые плагины из коробки
                            • vuepress-plugin-copy-code2: Обеспечивает функцию блокировки кода копирования одним щелчком мыши.
                            • vuepress-plugin-copyright2: Добавление информации об авторских правах при копировании или отключении копирования и выбора.
                            • vuepress-plugin-feed2: Поддержка каналов
                            • vuepress-plugin-md-enhance: Предоставляет больше синтаксиса Markdown
                            • vuepress-plugin-photo-swipe: Плагин предварительного просмотра изображений, основанный на фотопролистывании
                            • vuepress-plugin-pwa2: Расширенная поддержка PWA
                            • vuepress-plugin-reading-time2: Время чтения и количество слов
                            • vuepress-plugin-sass-palette: Плагин в стиле Sass для всех плагинов и тем
                            • vuepress-plugin-seo2: Плагин для улучшения SEO
                            • vuepress-plugin-sitemap2: Плагин карты сайта

                            Совет

                            Вот некоторые другие плагины, которые по умолчанию не включены в тему, вы можете включить их в соответствии со своими потребностями.

                            • vuepress-plugin-lightgallery: Плагин предварительного просмотра изображений на основе lightgallery
                            • vuepress-plugin-redirect: Перенаправления страницы
                            • vuepress-plugin-remove-pwa: Plugins to remove pwa
                            • vuepress-plugin-search-pro: Client search plugin
