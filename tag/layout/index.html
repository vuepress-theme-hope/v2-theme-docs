                                                              Blog homepage

                                                              vuepress-theme-hope allows you to enable a blog-style homepage.

                                                              You need to set layout: BlogHome and home: true in the frontmatter of homepage.

                                                              Mr.HopeAbout 1 minBlogBlogHomeLayout
                                                              Path Navigation

                                                              The theme adds Breadcrumb support, you can configure it using breadcrumb in page frontmatter and theme options

                                                              Without any config, a Breadcrumb that matches the theme color is displayed at the top of the page content to help the reader understand the document structure.

                                                              vuepress-theme-hope improves the default home page.

                                                              To use it, set home: true in page frontmatter. Any extra content after the YAML front matter will be parsed as normal Markdown and rendered after the features section.
