                                                              主题行为选项

                                                              hopeTheme() 接受可选的第二个参数，即行为选项。行为选项控制主题的行为。

                                                              行为选项也可接受一个布尔值:

                                                              • false: 意味着所有选项都设置为 false
                                                              • true: 意味着 { check: true, compact:true, custom :false, debug: false }
